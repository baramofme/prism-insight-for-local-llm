### 섹션별 상세:

#### 3.1 포트폴리오 (Portfolio)
- 헤더: "포트폴리오" + "접기" 버튼 (keyboard_arrow_up)
- "투자중" 라벨 (text-[#5f6368])
- 금액: "$19,653,380.00" (large, bold)
- 변동: "+$453,120.00 (+2.36%)" (녹색, ▲)
- 전체 영역 클릭 가능 (cursor-pointer, hover:bg-[#f8f9fa])

#### 3.2 관심 목록 (Watchlist)
- 헤더: "관심 목록" + info(ℹ) + edit(✎) + add(＋) + 접기(▲)
- info 버튼 툴팁: "이 목록에는 Google의 다른 서비스에서 팔로우하는 모든 종목 코드가 포함됩니다."
- 각 종목 row:
  - 좌측: ticker (text-[11px] bold) + full name (text-[10px] gray)
  - 우측: 가격 (tabular-nums) + 변동률 (색상, 화살표)
  - hover: bg-[#f8f9fa]
- 하단: "+ 종목 코드 추가" 버튼 (text-[#1a73e8])

#### 3.3 주식 업종 (Equity Sectors)
- 헤더: "주식 업종" + "접기" 버튼
- 테이블 컬럼 (정렬 가능):
  - 종목 코드 (Symbols) - sortable ▲
  - 추세 (Trend) - sparkline 차트
  - 변동 (방향 화살표 ▲/▼)
  - 가격 (Price) - sortable ▲
  - 변동 (Change)
  - 변동률(%) (% Change) - sortable ▲
  - 전일 종가 (Prev Close) - sortable ▲
  - 시가 (Open) - sortable ▲
  - 고가 (High) - sortable ▲
  - 저가 (Low) - sortable ▲
  - 거래량 (Volume) - sortable ▲
  - 시가총액 (Mkt Cap) - sortable ▲

### 상세 구성:

#### 4.1 탭 바
- "홈" (Home) - 활성화: 언더라인 (h-0.5, bg-[#1a73e8])
- "조사" (Research)
- Material Design 스타일 탭

#### 4.2 지역 탭 (Pill 버튼)
- 7개: 미국/유럽/아시아/중남미/통화/암호화폐/선물
- 활성화: bg-[#e8eaed] text-[#1f1f1f]
- 비활성화: text-[#5f6368] hover:bg-[#f8f9fa]
- 가로 스크롤 가능 (overflow-x-auto)
- 지역 변경시 지수 데이터 변경됨

#### 4.3 시장 지수 (Market Indices)
- 가로 스크롤 카드 리스트
- 각 카드: 지수명 (text-xs, text-[#5f6368]) | 값 (text-sm, bold) | 변화율 (색상)
- 각 지수에 sparkline 차트 표시
- border-r 로 구분

#### 4.4 AI 시장 요약 (Market Summary Cards)
- 아시아 지역 선택 시 표시 (4개 카드)
- 각 카드: Sparkles 아이콘 + 제목 + "N sites" 배지
- "Dive deeper with AI" 버튼 (Bot 아이콘 + 텍스트 + ChevronRight)
- Read more (접기/펼치기)

#### 4.5 뉴스 (News)
- 헤더: "More news stories" + "From sources across the web"
- 각 뉴스: 아이콘(Newspaper/Globe) + 제목 + 출처(source) + 시간
- "Show more" 버튼

#### 4.6 주식 테이블 (Stock Tables)
- 3열 그리드 (grid grid-cols-1 md:grid-cols-3)
- "Most Active" (with volume), "Daily Gainers", "Daily Losers"
- 각 row: 순위 | ticker+이름 | 가격 | 변동률 (색상)
